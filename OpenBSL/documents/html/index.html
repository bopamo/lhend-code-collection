Leo Hendrawan

OpenBSL is a simple, light-weight, open source Boot Strap Loader (BSL) for small 8/16/32 bits microcontroller device family which is designed and implemented as part in the main code memory.

The implementation of OpenBSL is inspired by various application notes from Texas Instruments for MSP430 BSL (Boot Strap Loader).

At the moment, OpenBSL supports the following compiler/target device:

  • MSP430G2553 on MSP-EXP430G2 Launchpad (IDE: Code Composer Studio)
